Webcycle Linkbuilding is een Wordpress Plugin om de automatische linkplaatsing door de Webcycle tool mogelijk te maken.

== Description ==

Webcycle Linkbuilding helpt met het verbeteren van de vindbaarheid van uw Wordpress blog of website met natuurlijke en gerelateerde backlinks.

Meldt je gratis aan op Webcycle.nl

Nooit meer handmatig links plaatsen op uw website dankzij ons geautomatiseerde systeem. Het vinden van linkpartners was nog nooit zo gemakkelijk.

* Automatische linkplaatsing dankzij onze unieke tool

* Selecteer zelf geschikte linkpartners voor uw website

* Vind gerelateerde links binnen diverse categorieÃ«n

* Werkt in combinatie met vele andere Wordpress Plugins

Volg ons op Facebook of Google+ om op de hoogte te blijven van de laatste ontwikkelingen en acties.

== Installation ==

1. Upload de inhoud naar je plugins map `/wp-content/plugins/`
1. Activeer de plugin via je Wordpress plugin manager
1. Vul de gegevens in onder de nieuwe menu item "Webcycle linkbuilding"
1. Gebruik de shortcode [webcycle linkbuilding="true"] op de aangegeven backlink pagina
